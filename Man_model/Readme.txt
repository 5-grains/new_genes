이 모델은 NewGenes 프로젝트에서 유전자 육성을 위한 웹 페이지 아바타를 만들기 위해 개발되었습니다.
이 모델은 Blendr로 작성되었으며 Unity에서 사용하기 위해 고안되었습니다.
이 모델은 실행할때 Unity 또는 Blend가 필요합니다.
이 모델은 체형_탈모_피부 유형으로 구성되어 있으며, 각 유전자의 특성을 나타내는 숫자가 앞에 붙어 있습니다. 체형은 -2에서 2까지 5단계로, 탈모는 0에서 4까지 5단계로, 피부 유형은 0에서 4까지 5단계로 분류됩니다.
피부는 텍스쳐 폴더 안에 PNG 텍스쳐파일로 존재하며 유니티에서 사용할 땐 따로 Matererial 파일로 적용시킵니다.
유니티에서 Create -> Material 로 New Material생성후 텍스쳐 파일 Albedo에 드래그로 적용 후 메테리얼을 해당 아바타 얼굴에 그래그해서 적용가능합니다.
Blender에서 텍스쳐를 불러올 땐 Texture paint 에서 왼쪽 상단 이미지 -> 열기로 텍스쳐 파일을 불러온 다음 Shading 에서 베이스 컬러에 적용가능합니다. (해당 메테리얼만 적용가능) 
모델의 액션들은 아직 들어가 있지 않은 초기 모델입니다.
이 모델들의 기초 -2_0_0~2_0_0은 샘플 파일에 모델 스크린샷이 들어있습니다.
본 작업은 모두 Blender 3.5.1 Unity 2021.3.24f1에서 작업되었습니다.
